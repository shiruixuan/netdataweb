netdata 仪表板选项

这些是浏览器设置。每位查看者都有自己的设置。它们不会影响您的 netdata 服务器的运行。
设置会立即生效，并会永久保存到浏览器的本地存储中（“焦点获取时刷新 / 始终刷新” 选项除外）。
若要将所有选项（包括图表大小）重置为默认值，请点击 此处。

何时刷新图表？
当设置为 聚焦时，如果页面 / 标签页未获得用户的焦点，图表将停止更新。当设置为 总是时，图表将自动刷新。设置为 聚焦 时，这样当此页面未获得你的焦点时，可降低浏览器的 CPU 占用率（并延长笔记本电脑和平板电脑的电池续航时间）。设置为 总是 以便在操作另一个窗口（即更改某些设置）时，让此窗口中的图表自动刷新。
要显示哪些维度？
当设置为 非零时，（在当前视图内）所有值都被设为零的维度将不会从 Netdata 服务器传输过来（除非图表的所有维度值均为零，在这种情况下，此设置不起作用 —— 所有维度都会被传输并显示）。当设置为 全部时，所有维度都将始终显示。将其设置为 非零，可减少 Netdata 与您的浏览器之间传输的数据量，降低浏览器的 CPU 占用率（要绘制的线条更少），并使图例更便于查看（图例中的条目更少）。
如何处理隐藏图表？
When set to 销毁时，不在浏览器当前视口内（位于页面可见区域之上或之下）的图表将会被销毁，并且当它们再次变得可见时会被重新创建。当设置为 隐藏时，不可见的图表将仅被隐藏起来，以此简化文档对象模型（DOM）并加快浏览器的运行速度。将其设置为 销毁，以降低浏览器的内存占用。将其设置为 隐藏，以便在页面滚动时更快地恢复图表。
页面滚动处理？
当设置为 同步时，图表会在滚动操作结束后立即检查其可见性。在运行速度较慢的计算机上，这可能会影响页面滚动的流畅性。若要在滚动结束时更新页面，可将其设置为 异步。将其设置为 同步 则在滚动时立即更新图表。设置为 异步 可以在运行速度较慢的计算机上实现更流畅的页面滚动。
使用哪种图表刷新策略？
当设置为 并行时，可见图表会并行刷新（所有查询会并行发送至 Netdata 服务器），并且会异步呈现。当设置为 顺序 时，图表会逐个依次刷新。如果您的浏览器能够处理（大多数现代浏览器都可以），就将其设置为 “并行”；如果您使用的是较旧 / 运行速度较慢的计算机，那就将其设置为 “顺序”。
我们要不要重新同步图表刷新呢？
当设置为 重新同步时，仪表板将尝试重新同步所有图表，以便它们能同时进行刷新。当设置为 尽力而为时，各个图表的刷新时间可能会与其他图表稍有差异。通常情况下，仪表板会开始并行刷新这些图表，但依据计算机的运行速度以及网络延迟情况，图表之间会开始出现些许时间差。将此项设置为 重新同步 会尝试在每次更新时对图表进行重新同步。将其设置为 尽力而为 可能会减轻浏览器和网络的压力。
在所有图表上同步悬停选择？
启用该功能时，在一个图表上进行的选择将自动在所有其他可见图表上选择相同的时间范围，并且所有可见图表的图例都将更新，以显示所选的值。禁用该功能时，只有获得用户关注的图表会被选中。启用它可更好地洞察数据。如果您使用的计算机运行速度很慢，实际上无法执行此操作，则可将其禁用。
你希望在哪里看到图例？
netdata 可以将图例放置在两个位置：图表的 下方 （默认位置）或者图表的 右侧 。
切换此设置将会重新加载仪表板。
使用哪种主题？
netdata 自带两种主题：深色（默认主题）和 白色。
切换此主题将会重新加载仪表板。
你需要帮助吗？
netdata 可以在某些区域显示一些帮助信息，以协助你使用仪表板。如果这些提示框让你觉得厌烦，可以使用此开关将它们禁用。
切换此开关将会重新加载仪表板。
平移和缩放时启用数据填充？
当设置为 填充 时，图表在可见区域的前后都会填充更多数据，从而给人一种整个数据库都已加载的感觉。这种填充只会在对图表进行首次平移或缩放操作之后才会发生（最初，所有图表都只有可见数据）。当设置为 不填充 时，即便在进行了首次平移和缩放操作之后，也只有可见数据会从 netdata 服务器传输过来。
在图表上启用贝塞尔曲线？
当设置为 平滑 时，支持该功能的图表库将会绘制平滑曲线，而非简单的直线来连接各点。
请记住动态图表库 dygraphs作为 Netdata 仪表板中的主要图表库，只能对折线图进行平滑处理，无法对面积图或堆叠图进行平滑处理。当设置为 粗糙时，该设置可以降低浏览器消耗的 CPU 资源。
这些设置会随着图表的更新逐步应用。若要强制应用它们，请立即刷新仪表板。
启用所选单位的自动换算？
当设置为 换算单位 时，显示的值将动态地进行换算（例如，1000 千比特将显示为 1 兆比特）。netdata 可以对以下原始单位进行自动换算：kilobits/s, kilobytes/s, KB/s, KB, MB, and GB. 当设置为 固定单位 时，所有数值都将使用 netdata 服务器所保留的原始单位来呈现。
使用哪种单位来表示温度？
设置仪表板的温度单位。
将秒数转换为时间格式？
当设置为 时间时，以 秒 为单位呈现数据的图表将显示为 DDd:HH:MM:SS的格式。当设置为 秒时，将直接展示原始的秒数。
